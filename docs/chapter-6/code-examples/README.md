# Isaac Sim Perception Pipeline Example

This script demonstrates a basic perception pipeline in Isaac Sim, including camera and LiDAR sensor setup, data collection, and basic processing.

## Requirements
- NVIDIA Isaac Sim
- Omniverse Kit
- Python 3.8+

## Usage
```bash
# Run the perception pipeline
python perception_pipeline.py
```

## Description
This example sets up a simulated robot with perception sensors in Isaac Sim and demonstrates how to collect and process sensor data for robotics applications.
