# Isaac Sim Perception Examples

This directory contains examples for implementing perception systems in Isaac Sim, including object detection, semantic segmentation, and sensor fusion.

## Examples Included

1. **object_detection.py** - Complete object detection pipeline in Isaac Sim
2. **semantic_segmentation.py** - Semantic segmentation implementation with visualization
3. **sensor_fusion.py** - Multi-sensor fusion combining camera and LiDAR data
4. **validate-examples.sh** - Validation script for testing the examples

## Requirements
- NVIDIA Isaac Sim
- Python 3.8+
- OpenCV, NumPy, PyTorch, SciPy

## Usage
```bash
# Run object detection example
python object_detection.py

# Run semantic segmentation example
python semantic_segmentation.py

# Run sensor fusion example
python sensor_fusion.py
```
